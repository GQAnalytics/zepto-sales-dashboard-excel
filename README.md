# Zepto Sales Dashboard | Microsoft Excel

## Project Overview

This project showcases an interactive Sales Dashboard developed in Microsoft Excel to analyze sales performance across different cities, product categories, customers, and time periods.

The dashboard enables business users to monitor key performance indicators (KPIs), identify trends, and make data-driven decisions using interactive filters and visualizations.
